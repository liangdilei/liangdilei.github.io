2025 CSP 游寄

- J

很顺的开题，但是写到 T4 的时候没劲往下写了？太累了吗？

J : 100 + 100 + 100 + 52 = 352

- S

天崩开局。

一开始没想出 T1 正解，然后想打 T1 的 dp 暴力，调了 1h 都没调出来。

然后转向 T2，打了 T2 的暴力分又回来调 T1，又调了 1h，还是没想出来。

最后想着偷点特殊性质，然后在特殊性质的引导下把 T1 的正解贪心打出来了。

然而此时时间只剩下了半小时。最后还把 T2 的 freopen 给注释掉了，虽然开了 long long，但是答案的最大值设置成了 INT_MAX，喜提 0pts。

S : 100 + 0 + 0 + 0 = 100

总结

J 组发挥还行，问题在于 S 组。

应该先把除了 T1 的暴力分、性质分都拿下，再留 1h 时间想 T1 正解。

想不出正解的时候不要着急打暴力，应该看看性质，用性质引导做题。

并且还要留 5min 时间检查文件输入输出。

这次是败在了考试技巧。
