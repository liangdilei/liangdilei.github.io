linux命令行常用命令

常规

  1. pwd：输出工作目录

  2. ls：列出文件夹中的文件

  3. cd 目录名：切换工作目录

  4. cp 文件1 文件2：将文件1复制到文件2

  5. mv 文件1 文件2：剪切文件1到文件2

  6. mkdir 文件夹名：创建文件夹

  7. touch 文件名：创建文件

  8. rm 文件名：删除文件

  9. rmdir 文件夹：删除空文件夹

  10. rm -r 文件夹：删除文件夹

  11. diff 文件1 文件2：比较文件1与文件2

    如果相同则无输出，否则将给出

    字母表示的意思如下

    a：add

    c：change

    d：delete

    数字1,2 表示在文件一的 ‘数字1’ 到 ‘数字2’ 行。

    数字3,4 表示在文件二的 ‘数字3’ 到 ‘数字4’ 行。

    ‘>’ 在a时表示要添加的字符串；在c时表示文件一和文件二的内容，以分隔线隔开。

    ‘<’ 表示要删除的字符串。

vi/vim中常用命令：

  1. vi/vim 文件名：用vi/vim打开文件

  2. esc：回到一般模式

  3. i：在光标处输入文字

  4. x：删除光标处文字

manjaro 发行版中的一些常用命令

  1. sudo pacman -Syu：系统更新

  2. sudo pacman -S 包名：下载包（可能不是最新版）

  3. sudo pacman -Sy 包名：下载包（从服务器下载最新版）

  4. sudo pacman -Scc：清空包缓存

  5. sudo pacman -Rs 包名：删除包并删除为使用的依赖

  6. sudo pacman -Rns 包名：删除包并删除用户配置文件（彻底删除）

  7. sudo pacman -Rns $(pacman -Qdtq)：删除未使用的依赖

  8. df -h：查看磁盘空间
