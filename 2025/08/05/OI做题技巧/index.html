OI做题技巧

做题步骤

  1. 读题

  2. 想题

  3. 证明自己的想法是正确的

  4. 写题

  5. 调题

  6. 交题

其中2-3步可以循环，其他部分需要保证一遍过！

想题想不出来怎么办？

抓住题目给的提示词

  1. 最大值/最小值：贪心、动态规划、二分。

  2. 图：最短路、并查集、最小生成树。

  3. 树：树形DP、贪心。

把玩样例

通常来讲，题目给的样例都是十分有用的，可以帮你排查错误，发现性质。

实在想不出来的时候，可以带着摆烂的心态玩玩样例，说不定就找到题眼了呢。

一些找题眼的方法

  1. 关注数据范围，找到那个特别的量，很可能就是出题人有意而为之，还可以推断出算法的时间复杂度最多是多少。

  2. 找出读题时让你感到奇怪的地方，也很可能就是出题人给你的提示。

  3. 从最简单的情况开始思考，简化问题，一步步将问题扩展，很可能得到答案。

  4. 枚举一个量，相多于多了一个条件。在这个条件下思考，说不定会有启发。

  5. 正难则反。可以试着反方向思考。

小技巧

  1. 虚点：当多个点之间相互连接，且边权相同时，可以将它们都连向一个虚点，边权为原边权/2。这样就可以以 $O(1)$ 的时间复杂添加或删除一个点啦。

  2. 当要判断两个字符数量是否相等时，可以把它们分别转化为 $1$ 和 $-1$，这样把字符加起来，结果是 $0$ 的话就相等。

  3. 当数学公式里出现 $max$ 或 $min$ 时，可以想想对答案贡献的成分有哪些。

  4. 将对答案有贡献的点分组，分成几组互不影响的集合，并对每组分别求解将答案累加。

  5. 可以删除序列中的某个数，并使得序列中不存在相邻的数，可以用dp求解。设 dp[i][0/1] 表示前 $i$ 个数满足条件，且删或不删第 $i$ 个数所需的最少操作数。
